# Visit Card

A visually designed digital business card in HTML/CSS.

## Context / Challenge
First exercise in front-end of my bootcamp: designing a digital business card to practice HTML and CSS skills.

## Technologies Used
- HTML5 / CSS3
- Flexbox / Grid
- Google Fonts 
- Deployment via GitHub Pages 

## Preview
[Follow-me!](https://alizee-dev.github.io/myvcard/)

## Features
- Identity section with name and profession
- Skills showcase
- Contact block with icons
- “Hire Me” badge floating over the photo
